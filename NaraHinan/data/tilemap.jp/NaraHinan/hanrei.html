Code for NARAについて

「奈良を知り、奈良を育む」という理念に基づき、ITを活用して慣らしをより良くしていくための団体です。 奈良市の特徴を活かし、地域活性化にも貢献すべく活動中です。

  • facebook: www.facebook.com/Code4Nara
  • Twitter: @Code4Nara

このアプリケーションについて

このアプリケーションは、奈良市の避難所オープンデータを用いてその場所をマッピングしたものです。
現在は避難所のポイントデータのみですが、将来的にはハザードマップ、活断層マップ、明治期の低湿地マップ等を一緒に閲覧できるアプリにしたいと考えております。
また、同じプログラムを利用した他地域他テーマでのアプリ作成等も可能です。
Code for NARAまでお問い合わせください。

ライセンス

このアプリケーションは以下の著作物を改変して利用しています。

  • 避難所データ、奈良市、クリエイティブ・コモンズ・ライセンス 表示 2.1

種別の定義

  • 一次避難所
  • 避難のための立退きを行った居住者、滞在者その他の者を避難のために必要な間滞在させ、又は自ら居住の場所を確保することが困難な被災した住民その他の被災者を一時的に滞在させるための施設
  • 一次避難地
  • 地震災害が広域に及ぶ場合の避難場所又は広域避難地への中継地
  • 広域避難地
  • 地震災害が広域に及ぶ場合の避難場所
